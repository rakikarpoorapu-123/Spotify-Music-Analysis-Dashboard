# 🎵 Spotify Music Analysis Dashboard

## 📌 Project Overview

This project analyzes Spotify music data using Python and Power BI to discover insights into song popularity, streaming trends, artists, albums, and playlist performance.

The dashboard provides an interactive way to explore Spotify data and understand music listening patterns.

---

## 📂 Dataset

The dataset contains information such as:

- Track
- Artist
- Album Name
- Release Date
- Spotify Streams
- Spotify Playlist Count
- Spotify Playlist Reach
- Spotify Popularity
- Track Score

---

## 🛠 Tools Used

- Python 3.11
- Pandas
- Matplotlib
- Power BI Desktop
- GitHub

---

## 📊 Dashboard Features

### KPI Cards

- Total Streams
- Average Track Score
- Average Popularity

### Visualizations

- Top 10 Artists by Streams
- Top 10 Songs by Streams
- Songs Released per Year
- Spotify Popularity vs Spotify Streams (Scatter Plot)

### Interactive Filters

- Artist
- Album
- Release Year

---

## 📈 Key Insights

- Most streamed songs belong to a small group of highly popular artists.
- Spotify popularity generally increases with stream count.
- Recent years show an increase in song releases.
- Playlist reach has a significant impact on total streams.
